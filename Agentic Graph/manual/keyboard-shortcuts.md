# Keyboard Shortcuts

## Agentic Graph

| Action   | Shortcut                                             |
|----------|------------------------------------------------------|
| Settings | <kbd>⌘</kbd><kbd>,</kbd> |

## File

| Action | Shortcut |
|----|----|
| New | <kbd>⌘</kbd><kbd>N</kbd> |
| New Window | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>N</kbd> |
| Open | <kbd>⌘</kbd><kbd>O</kbd> |
| Save | <kbd>⌘</kbd><kbd>S</kbd> |
| Save As | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>S</kbd> |
| Create Version | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>S</kbd> |
| Import watsonx Orchestrate | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>I</kbd> |

## Edit

| Action | Shortcut |
|----|----|
| Undo | <kbd>⌘</kbd><kbd>Z</kbd> |
| Redo | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>Z</kbd> |
| Cut | <kbd>⌘</kbd><kbd>X</kbd> |
| Copy | <kbd>⌘</kbd><kbd>C</kbd> |
| Paste | <kbd>⌘</kbd><kbd>V</kbd> |
| Select All | <kbd>⌘</kbd><kbd>A</kbd> |
| Delete | <kbd>⌫</kbd> or <kbd>⌦</kbd> |

## Arrange

| Action | Shortcut |
|----|----|
| Align Left | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>\[</kbd> |
| Align Right | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>\]</kbd> |
| Group | <kbd>⌘</kbd><kbd>G</kbd> |
| Ungroup | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>G</kbd> |

## Analysis

| Action | Shortcut |
|----|----|
| Analyze Architecture | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>A</kbd> |

## Canvas

| Action           | How                                   |
|------------------|---------------------------------------|
| Pan              | Two-finger trackpad scroll            |
| Zoom             | Trackpad pinch, or mouse scroll wheel |
| Marquee select   | Click and drag on empty canvas        |
| Add to selection | <kbd>⇧</kbd>+click      |
